# Login Form Project

A simple and modern **Login Form UI** built using **HTML, CSS, and JavaScript**.  
The project demonstrates clean front-end structure, responsive design, and basic form interaction.

This login interface includes a visual **mascot illustration beside the form** to enhance user experience and make the layout more engaging.

---

## 🚀 Features

- Clean and modern login form design
- Responsive layout
- Mascot illustration beside the login form
- Structured HTML, CSS, and JavaScript
- Beginner-friendly project
- Easy to customize

---

## 📂 Project Structure

login-form-project
│
├── index.html      # Main login page
├── style.css       # Styling for the login form
├── script.js       # JavaScript for form interaction
└── mascot.png      # Illustration displayed beside the login form
